# CuddlyCuddle

一个沉浸式的 AI 角色聊天应用，让你可以与任何角色进行对话。

## 功能特性

- 🤖 **AI 角色对话** - 与各种 AI 角色进行自然对话
- 🎭 **角色创建** - 上传参考图片，AI 自动生成完整角色设定
- 🎨 **沉浸式界面** - 精美的角色立绘展示，支持透明背景
- 🎬 **视频生成** - 角色可以生成视频回应
- 🔊 **语音合成** - 支持多种语音模型，让角色开口说话
- 📱 **移动端适配** - 支持触摸滑动、响应式布局
- 💬 **私信模式** - 专注的输入体验
- 🖼️ **素材库** - 管理角色的图片和视频素材

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Markdown

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 部署

构建后的文件位于 `dist` 目录，可以部署到任何静态托管服务。
